# Crash course: HTML, CSS, Tailwind, JS (your portfolio)

You learn by reading **real code** in the project root file [`index.html`](../index.html) and, when you want to experiment, by editing files in **`course/playground/`** only. Do not change `index.html` while following this course.

## How to work

1. Open [`index.html`](../index.html) in your editor and in the browser (double-click the file or use a local server if you prefer).
2. Work through [`lessons/`](lessons/) in order (`01` → `13`). Each lesson is short: goal, concept (3–5 sentences), pointers into `index.html`, and an optional hands-on step.
3. For hands-on steps, open the matching file under [`playground/`](playground/) in the browser, change one thing, save, refresh. Playgrounds use the same Tailwind v4 browser setup as your portfolio (`@tailwindcss/browser@4` + `<style type="text/tailwindcss">`).

## Pacing

After each lesson, note what felt easy or confusing (even “1–5”). That keeps the next step appropriately small.
